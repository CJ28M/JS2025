Variable Rules:

  • Variables must start with a letter, underscore (_), or dollar sign ($).
  • Variables cannot start with a number.
  • Variable names are case-sensitive (e.g., myVar and myvar are different).
  • Use descriptive names for better readability.
  • Avoid using reserved keywords (e.g., var, let, const, function).
  • Use camelCase for multi-word variable names (e.g., myVariableName)
  • Cannot contain a hyphen (-) or space.
