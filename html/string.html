JavaScript Strings

String Concatenation

The ability to add or join strings together
*Strings* var make = 'Toyota'; var model = 'prius'; var year = 2021; var myRide = make + ' ' + model; var myBraggingRight = 'I drive a '+ year + ' '+ myRide; console.log(myRide); console.log("Type: ", typeof myRide); console.log(myBraggingRight); **Console** Toyota Prius Type: string Value: I drive a 2021 Toyota Prius
To measure the length of a string:
*Length Property* var make = 'Harley Davidson'; var model = 'Iron 883'; var year = 2017; var myRide = make + ' ' + model; var myBraggingRight = 'I drive a '+ year + ' '+ myRide; console.log(myRide.length); **Console** 24

concat()

*concat()* text = "Hello" + " " + "World!"; text = "Hello".concat(" ", "World!"); console.log(text); **Console** "Hello World!"

indexOf()

The indexOf(), is a versatile tool used to find the index of the first occurrence of a specified value within either a string or an array. It returns the zero-based index where the value is found, or -1 if the value is not present.
*indexOf() Strings* let myString = "Hello World"; let index = myString.indexOf("o"); console.log(index); let index2 = myString.indexOf("World"); console.log(index2); **Console** Output: 4 Output: 6
"Milk" starts at 0 on the list. If not found then its -1. If you search for the second "Eggs" you must say starting from "3".
*indexOf() Arrays* const groceryList = [ "Milk", "Butter", "Eggs", "Chicken", "Steak", "Rice", "Corn", "Eggs", ]; console.log(groceryList.indexOf("Eggs")); console.log(groceryList.indexOf("Carrots")); console.log(groceryList.indexOf("Eggs", 3)); **Console** Output: 2 Output: -1 Output: 7
lastIndexOf() returns the index of the last occurrence of a specified value in a string.**Remember it's Capital sensitive.**
*lastIndexOf()* let text = "Hello planet earth, you are a great planet."; let result = text.lastIndexOf("planet"); console.log(result); **Console** Output: 36

replace()

string.replace(searchString, replacerString)
*replace()* let textInfo = "Mr Blue has a blue house and a blue car"; let results = textInfo.replace('blue', 'red') console.log(results); **Console** Output: Mr Blue has a red house and a blue car
string.replace(searchString, replacerFunction)
*replace()* let results = textInfo.replace("blue", (match) => { return match.toUpperCase(); }); console.log(results); **Console** Output: Mr Blue has a BLUE house and a blue car
string.replace(regexp, replacerString) g=global
*replace()* let results = textInfo.replace(/blue/g, 'red') console.log(results); **Console** Output: Mr Blue has a red house and a red car
"i" stands for insensitive flag. It ignores upper and lower case sensitivity
*replace()* let results = textInfo.replace(/blue/gi, 'red') console.log(results); **Console** Output: Mr red has a red house and a red car
string.replace(regexp, replacerFunction)
*replace()* let results = textInfo.replace(/blue/gi, (match) => { return match.toUpperCase(); }); console.log(results); **Console** Output: Mr BLUE has a BLUE house and a BLUE car

toString()

Returns the content of a string
*toString()* let text = "Hello World!"; let result = text.toString(); console.log(result); **Console** Output: Hello World!

toUpperCase()

*toUpperCase()* let text1 = "Hello World!"; let text2 = text1.toUpperCase(); console.log(text2); **Console** Output: HELLO WORLD!

toLowerCase()

*toLowerCase()* let text1 = "Hello World!"; let text2 = text1.toLowerCase(); console.log(text2); **Console** Output: hello world!
