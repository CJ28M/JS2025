JavaScript Data Types

Primitive / Value Types

*String* var firstName = 'Mary'; console.log("Value: ", firstName); console.log("Type: ", typeof firstName); **Console** Value: Mary Type: string
The "string" is 'Mary'.
*Number* var age = 27; console.log("Value: ", age); console.log("Type: ", typeof age); **Console** Value: 27 Type: number
The "number" is 27.
*Boolean* var recedingHairLine = false; console.log("Value: ", recedingHairLine); console.log("Type: ", typeof recedingHairLine); **Console** Value: false Type: boolean
The "boolean" is false or true.
*Undefined* var favFood; console.log("Value: ", favFood); console.log("Type: ", typeof favFood); **Console** Value: undefined Type: undefined
The "undefined" is the ;
*Null* var toiletPaper = null; console.log("Value: ", toiletPaper); console.log("Type: ", typeof toiletPaper); **Console** Value: null Type: object
The "null" is null;.

Reference Types / Structural Types
