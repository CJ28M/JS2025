The Array object, as with arrays in other programming languages, enables storing a collection of multiple items under a single variable name, and has members for performing common array operations.
*arrays* var tvshows = ["gold rush", "breaking bad", "the big bang theory"]; console.log(tvshows); **Console** 0: "gold rush" 1: "breaking bad" 2: "the big bang theory" length: 3
[] is the array
*arrays* var tvshows = ["gold rush", "breaking bad", "the big bang theory"]; console.log(tvshows[2]); **Console** the big bang theory
When requesting a value you enter the index
*arrays* var tvshows = ["gold rush", "breaking bad", "the big bang theory"]; var userSavedMovies = []; userSavedMovies.push('Minecraft'); userSavedMovies.push('Scary movie'); userSavedMovies.push('Room 1908'); console.log(userSavedMovies); **Console** 0:"Minecraft" 1:"Scary movie" 2:"Room 1908" length: 3
.push method takes a value and adds it to the end of the array
*arrays* var tvshows = ["gold rush", "breaking bad", "the big bang theory"]; var userSavedMovies = []; userSavedMovies.push('Minecraft'); userSavedMovies.push('Scary movie'); userSavedMovies.push('Room 1908'); userSavedMovies.pop(); console.log(userSavedMovies); **Console** 0:"Minecraft" 1:"Scary movie" length: 2
