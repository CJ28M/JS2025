while loop

*while Loops adjust last bottle* var bottles = 99; while (bottles > 0) { if (bottles !== 1) { console.log(bottles + " bottles of stuff on the wall"); } else { console.log(bottles + " bottle of stuff on the wall"); } bottles -=1; } **Console** 99 bottles of stuff on the wall ... ... ... 1 bottle of stuff on the wall
